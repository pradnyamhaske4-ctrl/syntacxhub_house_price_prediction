Syntecxhub_California_Housing_Prediction
📌 Project Overview
This project implements a Machine Learning Linear Regression model to predict house prices based on the California Housing dataset. The goal of this task is to demonstrate the end-to-end process of data loading, feature selection, model training, and performance evaluation.

🚀 Features
Data Analysis: Uses Scikit-Learn to fetch and process real-world housing data.

Predictive Modeling: Implements a Multi-variable Linear Regression model.

Feature Importance: Analyzes how specific factors like Median Income (MedInc) and House Age (HouseAge) impact the final price.

Data Visualization: Includes a scatter plot comparing Actual vs. Predicted values to visualize model accuracy.

🛠 Tech Stack
Language: Python

Libraries: * Pandas & NumPy (Data Manipulation)

Scikit-Learn (Machine Learning & Metrics)

Matplotlib (Data Visualization)

📊 Model Evaluation
The model evaluates performance using two key metrics:

R² Score (Accuracy): Measures how well the independent variables explain the variance in price.

RMSE (Root Mean Squared Error): Indicates the average distance between the predicted values and the actual values.
